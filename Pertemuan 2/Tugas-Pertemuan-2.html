Simple Moving Average (SMA) m = 4

m <- 4

# make train copies for SMA section (consistent naming)
train_ma.ts <- train_ts
test_ma.ts  <- test_ts

# SMA on training
sma_train <- SMA(train_ma.ts, n = m)                 # has leading NA (m-1)
sma_fitted <- c(rep(NA, m), head(sma_train, -1))     # one-step-ahead fitted

# Forecast for test: use last available SMA value (flat forecast)
sma_last <- tail(na.omit(sma_train), 1)
sma_forecast_vals <- rep(as.numeric(sma_last), length(test_ma.ts))
# numeric forecast (no tsp) for safe accuracy calc

Akurasi SMA (train & test)

# TRAIN (exclude NA)
ok_train <- which(!is.na(sma_fitted))
e_train_sma <- as.numeric(train_ma.ts)[ok_train] - as.numeric(sma_fitted[ok_train])

## Double Moving Average (DMA) m = 4

m <- 4
# compute S1 and S2 on training
S1 <- SMA(train_ts, n = m)
S2 <- SMA(S1,    n = m)

At <- 2 * S1 - S2
Bt <- (2 / (m - 1)) * (S1 - S2)

# fitted in-sample (lag 1)
At_lag <- c(NA, head(At, -1))
Bt_lag <- c(NA, head(Bt, -1))
dma_fitted <- as.numeric(At_lag + Bt_lag)

# forecast values numeric for test (avoid tsp metadata issues)
A_last <- as.numeric(tail(At, 1))
B_last <- as.numeric(tail(Bt, 1))
h <- seq_len(length(test_ts))
dma_forecast_vals <- A_last + B_last * h

Kesimpulan: Dari hasil analisis perbandingan metode peramalan, diperoleh bahwa Holt-Winters merupakan metode terbaik untuk data CO amatan 181360. Hal ini terlihat dari nilai RMSE dan MAPE pada data uji yang paling kecil dibandingkan dengan metode lain. Alasan utamanya karena Holt-Winters mampu menangkap komponen tren sekaligus musiman dalam data, sedangkan metode sederhana seperti SMA dan DMA hanya mampu meratakan fluktuasi jangka pendek, dan metode SES maupun Holt cenderung kurang fleksibel dalam menghadapi pola musiman. Dengan demikian, penggunaan Holt-Winters lebih sesuai untuk data deret waktu yang memiliki pola musiman yang jelas seperti pada kasus ini.
